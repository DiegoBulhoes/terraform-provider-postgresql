# Data Source (postgresql_query)

Executes a read-only SQL query and returns the results. This data source is useful for querying PostgreSQL catalog tables or running custom read-only queries.

## Example Usage

```terraform
data "postgresql_query" "connections" {
  database = "my_application"
  query    = "SELECT usename, client_addr, state FROM pg_stat_activity WHERE datname = current_database()"
}

output "active_connections" {
  value = data.postgresql_query.connections.rows
}
```
